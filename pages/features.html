Features

OEDE의 주요 기능은 자료 조합, 교수자 선택, 멘토링, 학습분석, 인증, 복지 연계, 글로벌 이동성으로 구성됩니다.

🧩

Block-based textbook

학습자는 여러 자료 블록을 조합하여 자신만의 수업교재를 구성합니다.

👥

Multiple instructors

학교교사, 외부교사, 대학생 멘토, 은퇴교사, AI 튜터를 선택할 수 있습니다.

💬

Mentor dialogue

멘토와의 대화를 통해 학습 경로를 조정하고 이해를 심화합니다.

🧭

Learning continuity

이주, 전학, 재난 상황에서도 학습을 계속할 수 있도록 연결합니다.

📊

Learning analytics

학습 성취와 자기 인식 차이를 분석하여 지원 수준을 조정합니다.

🛡️

Credential mobility

학습 이력과 성취를 학교·지역·국가 간 이동 가능한 기록으로 관리합니다.

🤝

Welfare integration

상담, 복지, 다문화 지원을 학습 지원과 함께 통합합니다.

🌐

Multilingual access

다국어·다문화 학습 맥락을 고려한 접근을 제공합니다.
