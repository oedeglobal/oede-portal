Welcome to OEDE

About OEDE

OEDE는 AI 기반 학습 자원, 분산형 교수설계, 학습분석, 학습자 참여, 교육복지, 학습인증 및 이동성, 글로벌 학습 네트워크를 하나의 구조로 연결합니다. 핵심은 기존 교육체계를 대체하는 것이 아니라, 학교·교사·멘토·지역사회가 함께 축적한 실천적 지혜를 지속 가능한 교육 자산으로 이어 주는 데 있습니다.

7-Layer Architecture

  • AI-Driven Adaptive Learning Content
  • Distributed Instructional Design
  • Multi-Level Learning Analytics
  • Learner-in-the-loop
  • Educational Welfare Integration
  • Learning Credential & Mobility
  • Global Learning Layer

아래 이미지는 OEDE의 주요 기능을 직관적으로 보여 주는 대시보드 예시입니다. 웹사이트에서는 원본보다 크게 보이도록 보정하여 방문자가 화면 내용을 읽고 이해할 수 있게 구성했습니다.

Personal Learning Hub

학습자는 자신의 학교, 교재, 교사, 멘토, 학습자료를 한 화면에서 확인하며 개인 학습 공간을 구성한다.

Student–Admin Overview

학생 화면과 관리자 화면을 병렬로 보여 주어 학습 지원과 관리가 동시에 연결되는 구조를 제시한다.

Learning Asset Combination Space

다중 교재, 교사 자료, 멘토 자원, AI 추천 자료를 블록처럼 조합하여 개인화된 학습 경로를 만든다.

Distributed Learning Space

여러 국가와 교사 자원을 비교하고 선택하며 분산된 교육설계 경험을 학습 자산으로 전환한다.
